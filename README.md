# Install:
<br>

* _Puedes clonarlo en el directorio que quieras:_

```sh
  git clone https://github.com/v7y3/fack-TOOL.git
```

* _Entra a la herramienta:_
```sh
  cd fack-TOOL
```
* _Ejecuta el instalador:_
```sh
   bash install.sh
```

* _Finalmente lanza el script:_

```sh
    bash fack-TOOL.sh
```
